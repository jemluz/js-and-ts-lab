Title: Work on next topic
When: Use this when you want the assistant to start the next book topic from the study map and follow repository guidelines.

Prompt:
lets work on next topic: [some book topic from chapter-list.md]
please follow instructions at docs/index.md

Notes:

- Replace the bracketed placeholder with a specific topic from [docs/chapter-list.md](docs/chapter-list.md).
- `docs/index.md` contains repository conventions and example structure; follow it when creating examples.
- Example usage: paste the prompt into the chat, replace the topic, and the assistant will create a small focused example file in the matching folder.
