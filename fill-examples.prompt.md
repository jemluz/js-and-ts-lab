Title: Fill specific .md file with related examples
When: Use this when you want the assistant to fill a specific `.md` file with examples related to its topic, following repository guidelines.

Prompt:
Complete [some .md file name] with examples based on [some .ts or .js file name]

Notes:

- The .md file should have same name as the .ts or .js file, but with .md extension. For example, if the .ts file is `data-storage/arrays/adding-elements.ts`, the .md file should be `data-storage/arrays/adding-elements.md`.
